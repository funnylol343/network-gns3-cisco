# academic-network-gns3-cisco
[Projeto Académico]: Projeto de uma rede corporativa para a empresa "Pastelinho", usando GNS3 para simular e configurar routers Cisco com OSPF, DHCP e NAT. Desenvolvido para a UC de Arquitetura e Protocolos de Comunicação (APC), do CTeSP de Instalações Elétricas e Automação na ESTGA-Universidade de Aveiro.
